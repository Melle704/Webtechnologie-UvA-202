# Chatbox implementation

TABLE sessions [
    INT uid,
    CHAR(11) session_id,
    TIMESTAMP creation_date
]

TABLE messages [
    INT id,
    INT uid,
    TINYTEXT msg, message of up to 255 characters
    TIMESTAMP date
]

LOGIN:
1. upload `PHPSESSID` to `sessions` table.
2. remove previous session ID from `sessions` table.
3. add session ID to `sessions` table.
4. load message log from `msgs` table.

ON MESSAGE SEND (client):
1. send POST request to `broadcast_message.php` containing the message using AJAX.

ON MESSAGE SEND (server):
1. determine whether the `PHPSESSID` cookie is set on POST request.
2. record date and add message to `messages` table.

CLIENT LOOP:
1. send GET request to `broadcast_message.php` using AJAX.
1. determine whether the `PHPSESSID` cookie is set on GET request.
2. append new messages to message log.
2. sleep for 0.5 seconds.
3. repeat.
